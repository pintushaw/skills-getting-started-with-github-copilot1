## Run backend tests

Install dependencies:

```bash
/usr/local/bin/python -m pip install -r requirements.txt
```

Run all tests:

```bash
/usr/local/bin/python -m pytest -q
```

Run a single test file:

```bash
/usr/local/bin/python -m pytest tests/test_signup.py -q
```
